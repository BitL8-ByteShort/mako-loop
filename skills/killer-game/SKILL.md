# Killer Game

Use this mode pack with `mako-loop`. Read `../../references/killer-game.md` before building or judging a game.

## Core Rule

Playable beats broad. Nail the moment-to-moment loop, control feel, feedback, failure/restart, and rendering proof before adding menus or meta progression.

## Required Proof

- Game renders nonblank.
- Controls work.
- Win/loss/failure or scoring state exists.
- Restart/reset works.
- Desktop and mobile framing are checked when browser-based.
